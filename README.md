# TokenCord
Discord User Control / dump info from discord account token.
supports tokenlogin. (add your chrome version's chromedriver.exe)

# Usage
FROM SOURCE:
1. run DISTOK.py
2. insert token
3. enjoy!

FROM EXE:
1. run DISTOK.exe
2. insert token
3. enjoy!
